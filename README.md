# spring-boot-starter-httpclient


### 说明

 > 基于 httpclient 的 Spring Boot Starter 实现

1. 扩展常用函数
2. 实现连接心跳检查
3. 实现不同响应处理实现

### Maven

``` xml
<dependency>
	<groupId>${project.groupId}</groupId>
	<artifactId>spring-boot-starter-httpclient</artifactId>
	<version>${project.version}</version>
</dependency>
```
